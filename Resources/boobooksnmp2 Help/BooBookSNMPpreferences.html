Boobooksnmp2


Preferences

Preferences are shown in a Tabbed View. All settings are saved and applied automatically upon change, there is no save button.

Query Table Time Field Format

Specifies the format for time display in the Query Results tabs.
Format is per http://unicode.org/reports/tr35/tr35-6.html#Date_Format_Patterns , default is: M/d/yy HH:mm:ss .

MIB Library

You add MIB files to Boobooksnmp2 MIB library, and select which ones you want loaded by into MIB Tree.
If you make changes to which files are to be loaded into MIB tree, use the "Reload Tree" toolbar icon to relaod the MIB tree so changes made take effect.
Use drag and drop to sort MIB files library. Order in which files are listed here is the order MIB files are read in by libnetsnmp.
